combination sum II

This problem is not hard. But to find an elegant solution, it is not easy. I used a vector to find all unique numbers and their count. But acutally it is not necessary, only three different situations need to be considered.

  1. The first number in the vector is always allowed;
    2, the current number is allowed even if it is equal to the previous number;
    3, the number that is not equal to the previous one is allowed;
